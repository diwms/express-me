## Useful docker commands

- Stop all running containers

```bash
docker stop $(docker ps -aq)
```
- Go to container 

```bash
docker exec -i -t 665b4a1e17b6 /bin/bash #hash
```

- List of all running containers

```bash
docker ps
```
